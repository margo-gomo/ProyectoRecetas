# Sistema de Prescripción y Despacho de Recetas

## Cómo ejecutar (explicación sencilla)

1. **Conectar la base de datos**  
   Asegúrese de que la base esté creada y accesible con las credenciales configuradas.

2. **Correr el servidor Backend**  
   Inicie el proyecto del servidor (backend) y verifique que quede escuchando en el puerto configurado.

3. **Correr el Frontend**  
   Inicie la aplicación cliente (interfaz). Con esto ya puede ingresar y usar el sistema.

### Para una simulación exitosa
- Estar en la **misma red** (backend en una máquina y el/los clientes en otra/s, apuntando a la IP del servidor), **y/o**
- Correr **dos instancias** del `Main` en la misma computadora para simular dos usuarios (por ejemplo, médico y farmacéutico) contra el mismo backend.
